Modernify

Used stacks: HTML CSS JavaScript

Modernify is a lightweight and modern CSS framework designed to simplify the process of building responsive and accessible web interfaces. It offers a clean design system with support for dark mode, customizable variables, and utility classes. Built with pure HTML and CSS, Modernify is easy to integrate into any frontend project.

KZK Bielki

Used stacks: HTML CSS JavaScript PHP

KZK Bielki was a realistic public transport simulation on ROBLOX, inspired by Polish municipal transit systems. It featured detailed bus routes, schedules, and dynamic passenger information displays. The project focuses on immersion through custom models and a lifelike urban environment. It was developed by me in 2022-2025.
